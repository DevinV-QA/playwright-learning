# Playwright Python QA Automation

This project is a learning-focused Playwright automation framework built using Python and the Page Object Model (POM).

## Tech Stack
- Python
- Playwright
- Pytest

## Structure
- pages/ — Page Object Models
- tests/ — Test cases
- utils/ — Shared helpers (future)

## Covered Scenarios
- Login page validation
- Valid login flow
- Invalid login error handling

## Purpose
This repository exists to:
- Practice professional automation structure
- Demonstrate QA automation fundamentals
- Serve as a reference project for interviews
